# TrustScore AI - Buildathon Google 2026

## 🎯 Vision
Transformer les traces numériques du secteur informel en opportunités financières. 

## 🚀 La Solution
Utilisation de **Gemini 3 Flash** pour analyser des captures d'écran d'historiques Mobile Money.
- **OCR Multimodal :** Extraction précise des transactions.
- **Scoring prédictif :** Évaluation de la solvabilité (0-100).
- **Inclusion financière :** Plan d'épargne automatisé généré par IA.

## 🛠️ Stack Technique
- **Modèle :** Google Gemini 3 Flash (via Google AI Studio)
- **Format de sortie :** JSON structuré
- **Input :** Image (Screenshot de transaction mobile)
